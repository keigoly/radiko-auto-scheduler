# Radiko Auto Scheduler 📻

設定したスケジュールに従って、Radikoの放送局を自動で切り替え・再生するChrome拡張機能です。

## ✨ 特徴
- 週間スケジュール管理（24時間365日対応）
- エリアフリー対応（全国の放送局を選択可能）
- PC時刻に同期した正確な自動切り替え
- 自動エラー回避＆再生リトライ機能

## 🚀 使い方
1. このリポジトリをダウンロード（ZIPで保存）して解凍します。
2. Chromeの `chrome://extensions/` を開きます。
3. 「デベロッパーモード」をONにします。
4. 「パッケージ化されていない拡張機能を読み込む」からフォルダを選択します。
