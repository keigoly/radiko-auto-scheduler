# 📻 Radikoかんたん予約スケジューラー

設定したスケジュールに従って、Radiko（ラジコ）の放送局を自動で開き、確実に再生を開始するChrome拡張機能です。
毎日のラジオ聴取を自動化し、聞き逃しを防ぎます。

## 📖 アプリ概要

「いつも聞いているあの番組、再生ボタンを押すのが面倒…」
「作業に集中していて、開始時間を過ぎてしまった…」

そんな悩みを解決するために開発されました。
曜日と時間を指定するだけで、ブラウザが勝手にRadikoを開き、選局して再生してくれます。

### ✨ 主な特徴
1.  **週間スケジュール管理:** 曜日・時間・放送局を指定して、繰り返し予約が可能です。
2.  **エリアフリー対応:** 北海道から沖縄まで、全国の放送局を選択できます。
3.  **強力な自動再生ロジック:**
    * 再生ボタンを自動検知してクリック。
    * **エラー自動回避機能:** 「放送局を選局してください」等のエラーが出ても、自動でリロードして再試行します。
    * **サイレントモード:** 再生処理は裏側で行われるため、作業の邪魔をしません。
4.  **プライバシー重視:** スケジュールデータはブラウザ内にのみ保存され、外部に送信されることはありません。

## 🛠 インストール方法

現在、Chromeウェブストアでの公開準備中、または開発者モードでの導入となります。

1.  このリポジトリ（または配布ファイル）をZIPでダウンロードし、解凍します。
2.  Google Chromeを開き、アドレスバーに `chrome://extensions/` と入力します。
3.  右上の **「デベロッパーモード」** をONにします。
4.  左上の **「パッケージ化されていない拡張機能を読み込む」** をクリックします。
5.  解凍したフォルダ（`manifest.json` が入っているフォルダ）を選択します。
6.  拡張機能リストに「Radikoかんたん予約スケジューラー」が追加されれば完了です！

## 📱 使いかた

1.  Chromeのツールバーにある「予」アイコンをクリックします。
2.  **「曜日」「時間」「エリア」「放送局」** を選択します。
3.  **「スケジュールに追加」** ボタンを押すとリストに保存されます。
4.  あとはブラウザを開いておくだけ。時間になると自動で新しいタブが開き、ラジオが流れます。

### ⚙️ 設定・その他
右上の歯車アイコンから設定メニューを開けます。
* **ダークモード:** 画面を暗い配色に切り替えます。
* **不具合の報告:** 動作しない場合やバグを見つけた場合は、専用フォームからご報告ください。
* **開発者を支援する:** もしこのアプリを気に入っていただけたら、開発者の活動をご支援いただけると励みになります。

## ⚠️ 注意事項・トラブルシューティング

* **PCがスリープ状態の場合:** ブラウザが動いていないため、自動再生は作動しません。
* **エラーが出る場合:** 本アプリはRadiko側の仕様変更などの影響を受ける可能性があります。もし再生されない場合は、手動でページをリロードするか、設定画面の「不具合の報告」よりお知らせください。
* **対応ブラウザ:** Google Chrome (最新版推奨)

---

## 🔒 プライバシーポリシー
本拡張機能は、ユーザーのスケジュール設定データをChromeのローカルストレージ（`chrome.storage.local`）にのみ保存します。
個人情報や閲覧履歴を外部サーバーへ送信・収集することは一切ありません。
